02 — Projects

Selected Work

A handful of projects I'm proud to have shipped.

DevFlow

A real-time developer productivity dashboard that aggregates GitHub activity, CI/CD pipelines, and team metrics into a single unified view. Features WebSocket live updates and customisable widget layouts.

  • TypeScript
  • Node.js
  • WebSockets
  • PostgreSQL

Lumina UI

An open-source component library built with pure TypeScript and CSS custom properties. Zero runtime dependencies, fully accessible (WCAG 2.1 AA), and tree-shakeable. Ships as ESM and CJS.

  • TypeScript
  • CSS
  • Rollup
  • Accessibility

TerraMap

An interactive geospatial data visualiser for climate and population datasets. Renders millions of data points at 60fps using WebGL-accelerated layers. Integrates with open REST APIs for live data feed ingestion.

  • D3.js
  • Mapbox GL
  • WebGL
  • REST APIs
